## Short preview blurb (READMEトップ / SNS共有文)

intentloop-harness は [intentops-harness](https://github.com/Da-P-AIP/intentops-harness)
の正常進化形。セッション内で閉じていた品質リファインループを、**セッションを跨いで永続化**する。
[axis-hfe](https://github.com/Da-P-AIP/axis-hfe)（6軸品質ゲート）と
[axis-thought-packet](https://github.com/Da-P-AIP/axis-thought-packet)（継続カプセル）を、
三層データ基盤＋RAGの上で配線し、宣言・手続き・エピソードの三系統記憶として運用する。

系譜：`axis-hfe` → `axis-thought-packet` → `intentops-harness` → **`intentloop-harness`**
